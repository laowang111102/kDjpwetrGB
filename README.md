# 个性化餐饮场所推荐系统 python568

## 项目概述

本项目是基于Python语言的个性化餐饮场所推荐系统，旨在提供用户友好的餐饮推荐服务。系统具有管理员管理、用户管理、餐厅信息管理以及系统管理等丰富功能。

## 技术栈

### 后端技术

- Python
- Django框架

### 前端技术

- VUE

### 开发工具

- PyCharm

## 功能模块

### 系统角色

- 管理员
- 普通用户

### 系统功能

- **系统首页**：展示餐饮场所推荐信息。
- **个人中心**：用户可以查看与编辑个人信息。
- **用户管理**：管理员可以对用户信息进行管理。
- **餐厅信息管理**：维护餐厅详细信息，包括餐厅类型。
- **餐厅类型管理**：分类管理各类餐厅。
- **留言板管理**：用户反馈与交流的平台。
- **系统管理**：系统设置与维护功能。

## 运行环境

- Python环境（建议版本3.x）
- 兼容的主流浏览器

## 部署步骤

1. 配置Python环境。
2. 安装Django框架。
3. 配置数据库。
4. 运行后端服务。
5. 构建前端页面。
6. 部署到服务器。
